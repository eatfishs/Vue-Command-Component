## 开始

通过你项目使用的包管理器进行安装


::: code-group

```bash [npm]
$ npm install -D vite
```

```bash [Yarn]
$ yarn add -D vite
```

```bash [pnpm]
$ pnpm add -D vite
```

```bash [Bun]
$ bun add -D vite
```

:::


多嘴一句,建议你使用`Anthony Fu`大佬的[`@antfu/ni`](https://www.npmjs.com/package/@antfu/ni)

```bash
npm i -g @antfu/ni
```

然后你就可以无视包管理器了
```bash
ni vue3-commadn-component
```
