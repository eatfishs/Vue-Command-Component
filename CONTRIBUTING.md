# 贡献指南

感谢您对Vue Command Component项目的关注！我们欢迎所有形式的贡献。

## 🚀 快速开始

### 环境要求
- Node.js >= 16
- pnpm >= 8
- Git

### 本地开发设置

```bash
# 克隆仓库
git clone https://github.com/slothvips/Vue-Command-Component.git
cd Vue-Command-Component

# 安装依赖
ni

# 启动开发模式
nr dev

# 启动文档网站
nr doc:dev
```

## 📖 项目结构

```
Vue-Command-Component/
├── packages/

└── package.json
```

## 🛠️ 贡献类型

### 1. Bug修复
- 在GitHub Issues中报告bug
- 提供复现步骤
- 提交修复PR

### 2. 新功能开发
- 先在Issues中讨论功能需求
- 获得maintainer确认后开始开发
- 遵循项目代码规范

### 3. 文档改进
- 修复文档错误
- 添加使用示例
- 翻译文档内容

### 4. 新UI库适配
- 参考现有适配器实现
- 提供完整的测试用例
- 更新相关文档

## 📝 开发规范

### 代码风格
- 使用TypeScript
- 遵循ESLint规则
- 使用Prettier格式化

### 提交规范
使用[Conventional Commits](https://conventionalcommits.org/)格式：

```
feat: 添加新功能
fix: 修复bug
docs: 更新文档
style: 代码格式调整
refactor: 重构代码
test: 添加测试
chore: 构建工具或依赖更新
```
